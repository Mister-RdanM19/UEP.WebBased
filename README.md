# 📂 Ubah Extension Pro (Web-Based)
### Developed by **Mr.Rm19**

![License](https://img.shields.io/badge/license-MIT-green)
![Language](https://img.shields.io/badge/language-HTML%2FCSS%2FJS-orange)


**Ubah Extension Pro** adalah sebuah tool berbasis web modern yang dirancang untuk mengubah ekstensi file secara cepat, aman, dan tanpa perlu mengunggah file ke server (Client-Side Processing). Tool ini hadir dengan antarmuka **Glassmorphism** yang futuristik dan database 500+ ekstensi file.

---


##  Fitur Unggulan

* **⚡ Instant Conversion:** Ubah ekstensi file apa pun dalam hitungan detik.
* **🔒 Privacy Focused:** File tidak pernah diunggah ke server. Semua proses terjadi langsung di browser Anda.
* **🔍 Smart Autocomplete:** Database internal dengan 500+ daftar ekstensi file populer untuk memudahkan pencarian.
* **🎨 Modern UI/UX:** Desain profesional dengan tema **Dark Mode** neon dan **Light Mode** yang lembut.
* **📱 Fully Responsive:** Bisa diakses dengan lancar melalui Desktop maupun Smartphone.
* **🖱️ Drag & Drop:** Dukungan penuh fitur tarik-lepas file untuk penggunaan yang lebih intuitif.

---

##  Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi web standar tanpa library eksternal yang berat, memastikan kecepatan loading yang maksimal:

* **HTML5** - Struktur aplikasi.
* **CSS3** - Glassmorphism design & Custom Animations.
* **Vanilla JavaScript** - Logika pemrosesan file (Blob) & Autocomplete.
* **Google Fonts** - Tipografi profesional (Orbitron & Poppins).
